Formato de Texto

Las primeras etiquetas que veremos son las que se usaban en el pasado para dar formato al texto. Para ello tenemos una serie de etiquetas que escribimos en HTML envolviendo la palabra o el texto y que transforman ese texto en el formato que nosotros le hayamos querido dar. Algunas de estas etiquetas están no recomendadas (deprecated) por lo que no debemos emplearlas. Otras etiquetas han adquirido un nuevo significado en las últimas versiones de HTML, pero no vamos a entrar a definir este nuevo significado. Debido a su amplia difusión en el pasado conviene conocer los que fueron usos tradicionales de estas etiquetas, a medida que avances en el conocimiento de HTML podrás comprobar cómo para algunas etiquetas se consideran hoy día nuevos significados.

ETIQUETAS USO OBSERVAACIONES

<strong>...</strong> Poner texto en negrita Puede ser sustituido por CSS
<i>...</i> Poner texto en cursiva Puede ser sustituido por CSS
<em>...</em> Poner texto en cursiva Puede ser sustituido por CSS
<u>...</u> Poner texto en subrayado Puede ser sustituido por CSS
<small>...</small> Poner texto mas pequeño Puede ser sustituido por CSS
<big>...</big> Poner texto mas grande Puede ser sustituido por CSS
<sub>...</sub> Poner texto subindice Puede ser sustituido por CSS
<sup>...</sup> Poner texto superindice Puede ser sustituido por CSS

<s>...</s> Poner texto como tachado Puede ser sustituido por CSS
<del>...</del> Poner texto como tachado Puede ser sustituido por CSS

Texto en Negrita:

<b>: Este elemento se utiliza para aplicar formato de negrita al texto. Es útil cuando se desea resaltar visualmente ciertas palabras o frases, pero no implica una importancia adicional para el contenido.
Hola Mundo...

Texto Importante:

<strong>: A diferencia de <b>, <strong> no solo proporciona formato en negrita, sino que también indica que el texto es de importancia significativa en términos de estructura y contenido. Los navegadores suelen representar el texto dentro de <strong> en negrita.
Hola mundo...

Texto en Cursiva:

<i>: Este elemento se utiliza para aplicar formato en cursiva al texto. Es comúnmente utilizado para destacar palabras o frases en un tono diferente o para expresar un cambio en el estilo del texto.
Hola mundo...

Texto Enfatizado:

<em>: Aunque visualmente similar a <i>, <em> se utiliza para enfatizar el texto de manera semántica. Los lectores de pantalla pueden interpretar <em> pronunciando las palabras con un énfasis adicional, lo que ayuda a transmitir la intención del autor.
Hola mundo...

Texto más Pequeño:

<small>: Este elemento reduce el tamaño del texto, lo que puede ser útil para información menos relevante o notas aclaratorias que no deben destacarse visualmente.
Hola mundo...

Texto Marcado:

<mark>: Permite resaltar o marcar parte del texto, proporcionando una forma de señalar información importante. Es especialmente útil para resaltar términos clave dentro de un párrafo.
Hola mundo...

Texto Eliminado:

<del>: Indica que el texto contenido ha sido eliminado o tachado. Los navegadores suelen representar el texto dentro de <del> con una línea horizontal a través de él.
Hola mundo...

Texto Insertado:

<ins>: Se utiliza para marcar el texto que ha sido insertado o agregado al documento. El texto insertado a menudo se subraya visualmente.
Hola mundo...

Texto en Subíndice:

<sub>: Coloca el texto en una posición de subíndice, útil para representar fórmulas químicas, expresiones matemáticas o cualquier contenido que deba aparecer ligeramente por debajo de la línea normal. Hola mundo...

Texto en Sobrescrito:

<sup>: Eleva el texto a una posición de sobrescrito, adecuado para representar notas al pie, exponentes o cualquier contenido que deba aparecer ligeramente por encima de la línea normal.
Hola mundo...
